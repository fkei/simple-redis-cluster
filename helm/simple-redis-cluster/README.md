# Setup

Redis Cluster 4.0.10

## Pre

- Docker needs to be running.
- Helm cli installed.

## QuickStart

helm install

```
make install
```

> docker image -> [https://hub.docker.com/r/fkei/simple-redis-cluster/](https://hub.docker.com/r/fkei/simple-redis-cluster/)

## Redis CLI Access


```
make cli
```

## docker build (developer)

to `make help`

## Help

```
make help
```
